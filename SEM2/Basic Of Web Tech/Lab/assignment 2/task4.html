Introduction of HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page and consists of a series of elements. HTML elements tell the browser how to display the content.

Text Formatting Tags

HTML provides various tags for formatting text, such as:

  • <b>- Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <u> - Underlined text
  • <mark> - Marked text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

Anchor Tag

The <a> tag defines a hyperlink. It is used to link to other web pages, files, locations within the same page, email addresses, or any other URL.

Example: <a href="https://www.w3schools.com">Visit W3Schools.com!</a> Visit W3Schools.com!

Image Tag

The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image, and the alt attribute provides alternative text for the image.

Example: <img src="image.jpg" alt="Description of the image">

Table Tags

HTML tables allow web developers to arrange data into rows and columns using the following tags:

  • <table> - Defines a table
  • <tr> - Defines a table row
  • <th> - Defines a table header cell
  • <td> - Defines a table data cell
